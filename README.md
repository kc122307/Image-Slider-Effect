# 🎯 Marvel 3D Image Slider

A modern and interactive **3D Image Slider** built using **HTML, CSS, and JavaScript**. The project features smooth transitions, animated content, and an elegant card-based carousel inspired by modern landing pages.

---

## 🚀 Live Preview

A visually engaging slider where the active card expands to fullscreen while the remaining cards are positioned as stacked previews on the side.

## ⚙️ How It Works

The carousel works by dynamically changing the order of elements inside the slider.

### Next Button

- Takes the first card
- Moves it to the end
- Automatically updates the layout

### Previous Button

- Takes the last card
- Moves it to the beginning
- Recreates the previous arrangement

This creates an infinite carousel effect without requiring complex animations or external libraries.
